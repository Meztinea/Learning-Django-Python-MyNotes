CLI 
La interfaz de línea de comandos o interfaz de línea de órdenes es un tipo de interfaz de usuario de 
computadora que permite a los usuarios dar instrucciones a algún programa informático o al sistema 
operativo por medio de una línea de texto simple.

Es una utilidad de línea de comandos (CLI) que permite interactuar con un proyecto Django de varias maneras.
Dentro del proyecto aparece como un script de python y ayuda con la administración del sitio. 
Con él podremos iniciar un servidor web en nuestro ordenador sin necesidad de instalar nada más, 
también permite crear aplicaciones, trabajar con bases de datos y empezar el desarrollo del servidor web.
